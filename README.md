# Spam Email Detection System

**Machine Learning Internship Final Project – SoftGrowTech**

High-accuracy (98%+) spam/ham email classifier using NLP (TF-IDF) + classical ML models (Naive Bayes, Logistic Regression, SVM, Random Forest, XGBoost).

### Features
- Full EDA + Word Clouds
- Professional text preprocessing pipeline (lemmatization)
- Multiple models with hyperparameter tuning
- Comprehensive evaluation (98%+ F1-score)
- Saved model + inference function
- Bonus: Streamlit web app for real-time testing

### How to Run
1. Place `spam_emails.csv` in `data/` folder
2. `pip install -r requirements.txt`
3. Open `spam_detection_notebook.ipynb` and run cells
4. (Optional) `streamlit run app.py`

**Results**: 98–99% accuracy on test set (see notebook).
